# 🐸 Android Bash Tools by GoldenFrog

Набор специализированных скриптов для продвинутых пользователей Android. Оптимизировано и протестировано на **POCO F3 (alioth)** на базе **Evolution X** (Android 15), но в целом совместимо с большинством устройств при наличии Termux, ADB или Root.

## ⚠️ Дисклеймер

Использование скриптов осуществляется на свой страх и риск.
Автор не несёт ответственности за возможные последствия.

## ⚙️ Требования

* Android 11+
* Root-доступ (Magisk / KernelSU)
* BusyBox или Toybox
* Termux / adb shell / root-terminal

## 🛠 Состав инструментов

### 📊 1. [cpu.sh](./scripts/cpu.sh) — Живой монитор системы
Профессиональная утилита для отслеживания состояния железа в реальном времени.
* **CPU**: Мониторинг частоты и нагрузки каждого из 8 ядер по отдельности.
* **GPU & Thermal**: Температура графического процессора и CPU из системных зон.
* **Battery**: Отображение уровня заряда, «здоровья» батареи, температуры и текущего тока в mAh.
* **Throttling**: Автоматическая проверка системы на наличие активного троттлинга.

### 📦 2. [apks.sh](./scripts/apks.sh) — Продвинутый установщик
Автоматизирует процесс установки большого количества приложений, экономя время и избегая ручной установки split-пакетов.
* **Split APKs**: Поддержка формата `.apks` через распаковку и системный вызов `pm install-multiple`.
* **Smart Logs**: Анализ ошибок установки (подписи, архитектура, память) с выводом рекомендаций по исправлению.
* **Clean-up**: Автоматическая очистка временных файлов после успешного завершения.

### 📺 3. [tv_tools.sh](./scripts/tv_tools.sh) — Менеджер Android TV
Управление телевизором или приставкой прямо с твоего смартфона через ADB по Wi-Fi.
* **Media Capture**: Снятие скриншотов и запись экрана с настройкой битрейта и длительности.
* **App Launcher**: Быстрый запуск системных утилит и установленного файлового проводника, например MT Manager, на удаленном устройстве.
* **Power Control**: Удаленная перезагрузка или выключение телевизора.

### 📦 4. [unpacker.sh](./scripts/unpacker.sh) — Распаковщик архивов
Инструмент для сверхбыстрой распаковки тяжелых ZIP-архивов (OBB, данные приложений) на Android.
* **Native Speed**: Использует скомпилированный бинарник `unzip`, обходя медленные Java-методы стандартных проводников.
* **Visual Progress**: Интерактивная шкала прогресса в терминале.
* **Safe Search**: Автоматический поиск и нумерованный выбор архивов в указанной директории.
* **Root Friendly**: Идеально подходит для работы в защищенных разделах `/data/`.

### 🔍 5. [monitor.sh](./scripts/monitor.sh) — Бортовой компьютер системы
Комплексный диагностический отчет, объединяющий состояние «железа» и системные логи в одном окне. Идеально подходит для быстрой проверки здоровья устройства.
* **Deep Storage & RAM**: Подробная статистика раздела /data (всего/занято/свободно) и мониторинг доступной оперативной памяти в МБ.
* **Battery Voltage Control**: Прямое чтение напряжения аккумулятора в mV. Незаменимо для устройств с изношенной батареей, когда проценты заряда отображаются некорректно.
* **Error Logging**: Автоматический анализ последних 500 строк logcat на наличие критических ошибок с выводом превью прямо в терминал.
* **Root & Security**: Честная проверка прав суперпользователя через UID и отображение статуса SELinux.
* **Smart Top**: Список самых ресурсоемких процессов с идеально ровным форматированием колонок.
 
Для более подробной документации по данному скрипту, вам стоит [заглянуть сюда](./monitor.md)

## 🚀 Установка и использование

1. Клонируйте репозиторий (`git clone https://github.com/goldenfrog-2577/Android-Bash-Tools.git`) или скачайте нужный скрипт.
2. Поместите скрипты в `/data/local/scripts/` (рекомендуется).
3. Установите права на выполнение:
   ```bash
   chmod +x *.sh
   ```
4. Для работы **cpu.sh** и **apks.sh** требуются права **Root**.
